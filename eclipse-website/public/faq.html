Frequently Asked Questions

Find answers to common questions about Eclipse Client

How do I install Eclipse?

Simply download the Eclipse.exe loader. It will guide you through the process. For the client to work, the Eclipse.jar must be placed inside your .minecraft/mods folder.

Which version of Minecraft do I need?

Eclipse is optimized for the Official Minecraft Launcher. Make sure you are running the version specified in the loader (e.g., 1.21.4).

Can I use other mods with Eclipse?

Yes, as long as they are compatible with the same mod loader (Fabric/Forge). Just drop them in the same mods folder.

The loader says "Minecraft Path Not Found"

This happens if you moved your game folder. Go to the "Settings" tab in the loader and manually select your .minecraft directory.

The mod doesn't show up in-game

Double-check that the Eclipse.jar is actually in your mods folder and that you have launched the correct profile from the Official Launcher.

My game crashes on startup

Ensure you don't have conflicting mods. Try running Eclipse on a clean .minecraft folder first.
